Hands-on Practice

Practical Lab

Practice slowly and repeat each activity until you can complete it without help.

01

Identify Computer Hardware Parts

Recognize monitor, keyboard, mouse, CPU cabinet, ports, printer, speakers, and storage devices.

02

Start and Shut Down Computer Properly

Learn correct startup, login, restart, shutdown, and basic safety practices.

03

Use Keyboard and Mouse

Practice typing, selecting text, right click, double click, drag and drop, and basic shortcuts.

04

Create Folder and Rename Files

Create folders, save files, rename files, copy files, and delete unnecessary files safely.

05

Use MS Word / LibreOffice Writer

Type a paragraph, format text, align content, add a heading, and save the document.

06

Use Spreadsheet Basics

Enter data in rows and columns, use simple formulas, and format a basic marks table.

07

Create a Simple Presentation

Create slides with a title, bullet points, images, and a clean theme for a small topic.

08

Use Internet Safely for Learning

Search academic topics, identify useful sources, avoid unsafe links, and do not share passwords.
